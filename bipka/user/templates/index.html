Сервис благотворительности для ресторанов

Данный сервис предначначен для ресторанов, которые готовы пожертвовать продукты питания благотворительным организациям. Многие продукты выкидаваются, потому что из них не успевают ничего приготовить. Это вредит экологии нашей планеты. Чтобы сократить количество мусора и помочь нуждающимся, воспользуйтесь данным сервисом!
